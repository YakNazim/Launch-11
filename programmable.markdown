# All FC Programmable Parts

A list of parts that have PSAS software on the Flight Computer:


 - Main Flight Computer (_Intel Atom N455 1.66 GHz x86_64_)
   - OS: Linux 3.10
 - GPS (_STM32_)
   - OS: ChibiOS
 - Video1 (_Raspberry Pi ARM11_)
   - OS: Linux
 - Video2 (_Raspberry Pi ARM11_)
   - OS: Linux
 - Control (_STM32_)
   - OS: ChibiOS
 - Data (_STM32_)
   - OS: ChibiOS
 - RNH (_STM32_)
   - OS: ChibiOS
